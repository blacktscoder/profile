```bash

```bash

```

```

---

## Writing Content

### Blog posts

Add a new `.mdx` file to:

```
app/blog/posts/
```

Each post supports frontmatter like:

```yaml
---
title: "Post title"
description: "Short summary"
date: "2025-02-01"
---
```

### Projects

Project write-ups live in:

```
app/work/projects/
```

These are written as reflective articles rather than technical documentation.
